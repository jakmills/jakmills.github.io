About Me

Projects

  • Project A (BlueWave): Built a Chat Application for ASRC federal in my Software Engineering 1 class. This application uses JavaFX for the front end, and uses Valkey for the backend which allows users to message each other. We are planning on implementing a database to keep track of users and their messages. This project uses JavaFX, Java, Valkey, MySQL, Maven, Valkey Cache, and Google's Protobuff.
  • Project B (Jash Sports): Built a web app that shows users past, current, and scheduled professional soccer matches. There is also a section that shows user's relevant news articles pertaining to all big leagues. Users also have the ability to search and favorite teams, as long as their are logged in. This project was built with Firebase, HTML/CSS/JS, and Java. Javalin was used to connect the frontend to the backend
  • Project C (Zerra): Building a file sharing web app, similar to google drive. User's will be able to upload files, and share them with their friends. This web app is being developed using React for the frontend and Spring Boot Web for the backend. We are also using Hibernate as well as Lombok.
